# Инструкция по запуску EcoReport

## Начало работы

### 1. Подготовка
Склонируйте репозиторий:
```
git clone https://github.com/moevm/mse1h2026-ecoreport.git 
```
Откройте терминал и перейдите в директорию проекта:
```
cd mse1h2026-ecoreport
```
Заполните env файлы в соответствии с примером в файлах env.example
 
### 2. Запуск приложения
Выполните команду:
```
docker-compose up -d --build
```

### 3. Остановка приложения
Чтобы остановить все сервисы:
```
docker-compose down
```

## Первый запуск
1. Откройте http://localhost:8080/register в браузере
2. Пройдите регистрацию
3. Авторизуйтесь
4. Начните создавать отчеты
