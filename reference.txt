Модуль входных данных

Модуль содержит Pydantic-модели, описывающие структуру входных данных для генерации отчётов по экологической безопасности дренажных систем.
--------------------
Назначение

Модели обеспечивают:
- Спецификацию полей, необходимых для формирования отчёта.
- Автоматическую валидацию типов, диапазонов и форматов данных.
- Сериализацию/десериализацию JSON.
--------------------
Установка

Модуль не требует отдельной установки, достаточно скопировать файл `input_models.py` в проект. Зависимости:

- Python 3.9+
- Pydantic 2.x

Установка Pydantic: pip install pydantic
