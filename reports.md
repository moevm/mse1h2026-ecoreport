## О проекте
**EcoReport** — веб-сервис для автоматизированной генерации экологических отчётов

Сервис позволяет:
- вводить данные вручную;
- загружать данные из `.csv` / `.xlsx`;
- формировать отчёт в формате PDF (с возможностью расширения до DOCX);
- экспортировать данные в формате GeoJSON для использования в ГИС-системах;
- скачивать готовые файлы.

Проект реализуется в формате веб-приложения.

### Постановка задачи
Разработать веб-сервис, который:

1. Позволяет вводить данные:
   - вручную через форму;
   - через загрузку `.csv` / `.xlsx`.

2. Выполняет:
   - проверку типов данных;
   - проверку диапазонов;
   - проверку обязательных полей;
   - проверку формата файлов.

3. Генерирует:
   - отчёт в формате PDF (минимум);
   - при расширении — DOCX;
   - GeoJSON для ГИС-систем.

4. Позволяет:
   - скачать отчёт;
   - скачать GeoJSON;
   - очистить форму;
   - редактировать загруженные данные.

# Функциональные требования

## Ввод данных

Форма содержит блоки:
- Общая информация об объекте;
- Специальная информация (опционально);
- Материал;
- Результаты выщелачивания.

Дополнительно:
- Возможность загрузки табличного файла;
- Поддержка большого количества колонок.

---

## Валидация

- Проверка числовых значений (float);
- Проверка допустимых диапазонов;
- Контроль обязательных полей;
- Проверка формата файла;
- Отображение понятных сообщений об ошибках.

---

## Генерация отчёта

- В формате PDF или DOC;
- Соответствие нормативным документам;
- Наличие минимум одной столбчатой диаграммы;
- Отображение статуса генерации.

---

## Экспорт в GeoJSON

- Формирование GeoJSON-файла;
- Поддержка координат;
- Цветовая градация точек в зависимости от значения;
- Совместимость с ГИС-системами.

---


# Нефункциональные требования

- Простой и интуитивный интерфейс;
- Минимальная когнитивная нагрузка;
- Понятные сообщения об ошибках;
- Работа в браузере без установки ПО;
- Отсутствие ограничений по стеку технологий.
